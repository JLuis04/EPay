# Dashboard Administrativo Fintech

Frontend moderno inspirado en una operacion financiera/medico-administrativa, con enfoque en productividad, monitoreo y control transaccional.

## Stack

- React + TypeScript
- TailwindCSS
- Recharts
- Componentes reutilizables y datos mock

## Ejecutar

1. Instala Node.js (version 18+ recomendada).
2. Instala dependencias:

```bash
npm install
```

3. Inicia el entorno de desarrollo:

```bash
npm run dev
```

4. Build de produccion:

```bash
npm run build
```

## Estructura

- `src/components/layout`: sidebar y topbar
- `src/components/common`: cards, tabla con filtros, badges, alertas y detalle
- `src/components/charts`: graficas de lineas y barras
- `src/data/mockData.ts`: datos mock de KPIs, series y transacciones
